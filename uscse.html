Sistem Manajemen Laporan Kerja PLN Galesong

üìñ Penjelasan Alur Use Case

Aplikasi PLN Galesong merupakan sistem manajemen laporan kerja yang dirancang untuk mengelola aktivitas kerja harian karyawan dengan dua peran utama, yaitu Atasan (Admin) dan Karyawan (Kelompok). Alur kerja sistem dimulai dengan proses Login dimana kedua aktor (Atasan dan Karyawan) harus melakukan autentikasi terlebih dahulu dengan memasukkan username dan password untuk mengakses sistem. Setelah berhasil login, Atasan dapat mengakses dashboard untuk melihat ringkasan aktivitas, kemudian melakukan Manajemen Kelompok & Karyawan untuk mengelola data kelompok dan karyawan. Atasan juga dapat melakukan Monitoring Laporan untuk memantau semua laporan yang masuk, serta mengakses Statistik dan Generate Prediksi untuk analisis data dan prediksi kegiatan. Untuk keperluan pelaporan, Atasan dapat melakukan Export Data dan Manajemen Excel untuk mengelola file Excel. Di sisi lain, Karyawan dapat mengakses dashboard kelompok mereka, melakukan Input Laporan untuk memasukkan laporan kerja harian, melakukan Generate Prediksi untuk menghasilkan prediksi kegiatan, dan melakukan Export Data Kelompok untuk mengekspor data kelompok mereka ke format Excel. Secara keseluruhan, sistem ini memfasilitasi alur kerja yang terstruktur dari input data oleh karyawan, pemantauan dan analisis oleh atasan, hingga menghasilkan prediksi dan laporan yang dapat digunakan untuk pengambilan keputusan yang lebih baik.

üìã Deskripsi Use Case

üîê Use Case: Login (Digunakan oleh Semua Aktor)

Login

Baik Atasan maupun Karyawan harus melakukan proses login terlebih dahulu dengan memasukkan username dan password untuk mengakses sistem. Setelah berhasil login, sistem akan mengarahkan pengguna ke dashboard sesuai dengan peran mereka masing-masing (Atasan atau Karyawan).

üë§ Actor: Atasan (Admin)

1. Dashboard

Atasan dapat melihat dashboard dengan statistik lengkap tentang laporan, karyawan, dan kelompok

2. Manajemen Kelompok & Karyawan

Atasan dapat mengelola data kelompok dan karyawan, termasuk membuat, membaca, memperbarui, dan menghapus data kelompok kerja dan data karyawan

3. Monitoring Laporan

Atasan dapat melihat, memfilter, dan memantau semua laporan kerja dari semua kelompok

4. Statistik

Atasan dapat melihat statistik detail dan grafik analisis data untuk evaluasi dan pengambilan keputusan

5. Generate Prediksi

Atasan dapat menghasilkan prediksi kegiatan berdasarkan data laporan dan analisis historis

6. Export Data

Atasan dapat mengekspor data ke format Excel untuk keperluan laporan dan dokumentasi

7. Manajemen Excel

Atasan dapat mengunggah, mengelola, dan mengunduh file Excel untuk keperluan sistem

üë§ Actor: Karyawan (Kelompok)

1. Dashboard

Karyawan dapat melihat dashboard kelompok dengan statistik dan ringkasan aktivitas kelompok mereka

2. Input Laporan

Karyawan dapat membuat, melihat, mengedit, dan menghapus laporan kerja harian kelompok mereka

3. Generate Prediksi

Karyawan dapat menghasilkan prediksi kegiatan berdasarkan data laporan kelompok mereka

4. Export Data Kelompok

Karyawan dapat mengekspor data laporan kelompok mereka ke format Excel untuk keperluan dokumentasi dan pelaporan internal
